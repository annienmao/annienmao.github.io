Startup Setup

# overwrite default passowrd
passwd

# update apps
sudo apt update && apt upgrade

# start ssh and set it to start on boot
sudo systemctl start ssh
sudo systemctl enable ssh

Jupyter Lab Setup

sudo apt install python-pip
sudo apt-get install python-wheel
sudo pip install jupyterlab

Set a hashed password, and store it in ~/.jupyter/jupyternotebookconfig.py. Start the notebook: 

nohup jupyter lab --ip <local-ip> --port 8888 &

Install corss-compiler

sudo apt install mingw-w64

Install wine

When running exploit built for windows: 

sudo apt install wine
