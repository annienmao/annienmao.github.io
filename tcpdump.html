tcpdump

tags

Overview

tcpdump can be used to inspect the content of .pcap file. 

tcpdump -r <file>.pcap

tcpdump can also capture the network packets. 

 sudo tcpdump -i any -nX port <port>

To get all the packets after initial handshake and print out content in ascii. 

sudo tcpdump 'tcp[tcpflags] & (tcp-ack|tcp-push) != 0' -nA -r <file>.pcap
